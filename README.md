# ArduinoJoystick
Easy to use Library for Arduino Joysticks
Get floating point Numbers instead of analogReadings. 

# Wiring
 
 
 |Arduino|         Joystick|
 |----------|--------------|
 |5V        |      5V |
 |GND       |      GND |
 | A2        |      VRx |
 | A3        |      VRy |
 | D2        |      SW |
